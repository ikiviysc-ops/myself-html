# IKIVIY - 个人创作网站

IKIVIY 是我的个人创作网站，展示我开发的各种有趣项目和创意作品。这是一个不断成长的平台，我会在这里分享我的技术探索、创意实验和个人作品。

## 关于本网站

这个网站是我个人的数字空间，用于展示我在编程、设计和创意方面的探索。我计划在未来不断丰富内容，添加更多有趣的项目和作品，包括但不限于：

- 前端开发实验
- 交互设计作品
- 创意编程项目
- 技术学习笔记
- 个人作品集

## 技术栈

### 前端技术
- **HTML5 + CSS3 + JavaScript** - 基础技术架构
- **CSS 3D Transforms** - 实现立体视觉效果
- **IntersectionObserver API** - 优化滚动和动画性能
- **原生 JavaScript** - 无框架依赖，轻量高效

### 核心文件
- [trae_v2.html](file:///workspace/trae_v2.html) - 主页面，包含完整的界面和功能

## 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/ikiviysc-ops/myself-html.git
   cd myself-html
   ```

2. **本地预览**
   直接在浏览器中打开 `trae_v2.html` 文件即可查看效果。

3. **部署到 GitHub Pages**
   项目已配置自动部署到 GitHub Pages，推送到 `main` 分支后会自动更新。
