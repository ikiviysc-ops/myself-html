# Trae - 你的 AI 开发工具包

多智能体并行协作，从设计器到自定义代码全流程覆盖。

## 功能特性

- **完整智能体控制权** - 集成 MCP 协议，多智能体模式让你同时在多个实例并行工作
- **内嵌知识库** - 从 GSAP 动画、自定义组件到页面转场与 WebGL，全部内置
- **部署流水线** - 一键部署，持续集成，即时回滚，CDN 全球加速分发

## 技术栈

- HTML5 + CSS3 + JavaScript
- CSS 3D Transforms
- IntersectionObserver API
- 原生 JavaScript（无框架依赖）

## 快速开始

直接在浏览器中打开 `trae_v2.html` 即可预览效果。
